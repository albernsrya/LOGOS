## Optimization Package for Capital Investments 

### Problems that can be solved by this package:

- Knapsacks Problems include:
  Single Knapsack Problem, Bounded Knapsack Problem, Multiple-Knapsack Problem,
  Multi-dimensional Knapsack problem, Multi-choice Knapsack problem,
  Stochastic Programming of Multiple-Knapsack problem with uncertainties

### Installation:
./build.sh

### How to run:
- source activate riam_libraries
- python main.py -i inputfile.xml -o outputfile.o
